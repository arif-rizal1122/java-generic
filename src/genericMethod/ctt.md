# GENERIC METHOD

째 Generic parameter type tidak hanya bisa digunakan pada class atau interface
째 Kita juga bisa menggunakan generic parameter type di method
째 Generic parameter type yang kita deklarasikan di method, hanya bisa diakses di method tersebut, tidak bisa digunakan di luar method
째 Ini cocok jika kita ingin membuat generic method, tanpa harus mengubah deklarasi class
